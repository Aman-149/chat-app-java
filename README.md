# Java Chat Application

A simple **multi-client chat application** built with **Java Swing** for the client and **Java Sockets** for the server. Supports real-time messaging between multiple clients over TCP.

---

## Features

- GUI client using **Java Swing**
- Multi-client support via **threads**
- Real-time message broadcasting
- Enter messages via **text field** or **Send button**
- Connects to a server running on `localhost` or custom IP

---

## Requirements

- Java JDK 8 or higher
- IDE or terminal to compile and run Java programs

---
